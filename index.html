Games For Everyone

We like to focus on games & ideas that have universal appeal. This means that while designing a game, It distinctly appeals to kids & adults. Can this be done perfectly? No. But it's something we like to work on. Think about how you can watch a Disney animated movie as a kid as well as an adult.

Nostalgia

Game design & trends are constantly changing. We want to rejuvinate some old school gaming nostalgia. Hard levels & simple graphics which everyone can appreciate.

High Quality

We believe in making quality games that are robust and work not only on flagship devices, But on budget smartphones. This means writing clean optimized code and packaging it to be compatible with all devices.
